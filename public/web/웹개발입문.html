<!--주석문 태그-->

HTML5 페이지의 필수 태그

<!DOCTYPE html> - HTML5 문서임을 알리는 지시어
<html>, <head>, <title>, <body> 태그

HTML 태그 구성

  • 시작태그와 종료태그가 모두 있는 경우
    • <html> ... </html>, <title> ... </title>
  • 시작태그만 있는 경우
    • <br>

    태그와 속성은 대소문자 구분 없음

    <HTML> ... </html>

    속성 값에 불필요한 공백 - HTML5 표준에 어긋남

    웹 페이지 타이틀 달기

    <!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>여기에 타이틀 입력</title>
</head>
<body>
...

    <h1>, ..., <h6> 태그로 문단 제목 달기

      <h1> 태그

      <h2> 태그

      <h3> 태그

      <h4> 태그

      <h5> 태그
      <h6> 태그

    title 속성으로 툴팁 달기

    <h1 title="여기에 툴팁을 입력">

    텍스트 위에 마우스를 올리면 툴팁이 출력됨

    <p>로 단락 나누기

    <hr> 태그로 수평선 긋기

    종료 태그 </hr>를 사용하지 않는다

    <br> 태그로 새로운 줄로 넘어가기

    종료 태그 </br>를 사용하지 않는다

    문자, 기호, 심볼 입력

    < 는 &lt; 로 입력

    <pre> 태그로 개발자의 포맷 그대로 출력

    <pre> 태그를 사용하면
    사용자가 입력한 그대로 출력됨

    텍스트 꾸미기

    <b> 진하게
    <strong> 중요한
    <em> 강조
    <i> 이탤릭으로 강조
    <small> 한 단계 작은 문자
    <del> 삭제
    <ins> 추가
    <sup> 윗첨자
    <sub> 아래첨자
    <mark> 하이라이팅
